Вопросы:

  1. Предмет и задачи инженерной психологии.
  2. Характеристика систем «человек-машина-среда» (СЧМС).
  3. Особенности деятельности человека-оператора в СЧМС.
1 вопрос

Инженерная псхиология – это отрасль психологической науки, которая изучает закономерности информационного взаимодействия человека и техники в системах «человек-машина-среда».

Предпосылки возникновения инженерной психологии:

  1. Недостаточная эффективность СЧМС, их высокая аварийность, вызванная неудовлетворительным учетом в констуркциии этих систем функциональных возможностей человека.
  2. Рост травматизма людей, взаимодействующих с техническими системами.
  3. Большая текучесть кадров.

Инженерная психология имеет двойственный характер:

  1. Изучает психологические особенности человека.
  2. Имеет выраженный технический характер.

Важной задачей инженерной психологии является решение вопроса о распределении функций между человеком и техникой в СЧМС. Функции распределяются по принципу преимущества возможностей.

Преимущественные возможности для техники:

  • скорость выполнения вычислительных операций и просмотр многочисленных вариантов.
  • большой объем памяти и скорость извлечения информации.
  • стабильность в выполнении однообразных действий.
  • выполнение операций по строго заданному алгоритму.
  • нечувствительность к влиянию социальной среды.

Преимущественные возможности человека:

  1. Способность вырабатывать индивидуальный стиль деятельности.
  2. Способность принимать информацию по различным сенсорным каналам одновременно.
  3. Способность к формированию образов внешнего мира (воображение, фантазии, мечты).
  4. Способность генерировать идеи.
  5. Способность к решению не четко сформулированной задачи.
  6. Способность к предвидению событий (интуиция).
  7. Способность находить нестандартные решения в экстремальных ситуациях.
2 вопрос

Система СЧМС включают в себя:

  1. Техническое устройство
  2. Человек-оператор – это человек, осуществляющий трудовую деятельность посредством информационных систем и органов управления.

СЧМС классифицируются:

  1. По характерстикам человека:
    1. Моносистемы,
    2. Полисистемы.
  2. По типу взаимодействия человека и техники:
    1. Непрерывного взаимодействия (мясорубка).
    2. Эпизодического взаимодействия (контроль через некоторое время).
  3. По целевому назначению:
    1. Управляющие (водитель, летчик).
    2. Обслуживающие (автомойка).
    3. Обучающие (тренажеры, имитаторы).
    4. Информационные (компьютер, интернет).
    5. Исследовательские (метеозонд).
  4. По характиристками техники:
    1. Инструментальные (перфоратор, молоток).
    2. Простые (один оператор, одна функция системы).
    3. Сложные (многооператорность, несколько взаимосвязанных систем).
    4. Системотехнические комплексы (АЭС).

Характеристики качества деятельности человека-оператора в СЧМС:

  • Надежность (безошибочность),
  • Быстродействие (время реагирования),
  • Точность,
  • Безопасность.
3 вопрос

Особенности деятельности человека-оператора в СЧМС:

  1. Передача ряда функций автоматическим устройствам, что повышает значимость контроля за их работой.
  2. Кодирование информации на средствах ее отображения, что затрудняет формирование адекватного образа объекта и ситуации.
  3. Опосредованное восприятие внешнего мира и управляемого объекта с помощью информационной модели.
  4. Управление разнокодовой информацией.

Из этого следует: повышенная подверженность человека-оператора информационному стрессу.

Этапы деятельности человека-оператора:

  1. Прием информации (формирование перцептивного образа).
  2. Оценка и переработки информации (формирование оператвивного образа).
  3. Принятие решения (формирование последовательности целесообразных действий).
  4. Реализация принятого решения (использование выходных каналов человека – двигательного или речевого).

Средства операторской деятельности (рабочее место человека-оператора):

  1. Cредства отображения информации (СОИ)
  2. Органы управления (ОУ)
  3. Вспомогательные средства

Средства отображения информации:

  1. По модальности сигнала:
    1. Визуальные,
    2. Акустически,
    3. Тактильные.
  2. По форме сигнала:
    1. Абстрактные,
    2. Изобразительные.
  3. По функции выдаваемой информации:
    1. Командные,
    2. Ситуационные
  4. Способы использования показателей:
    1. Для контрольного чтения,
    2. Для качественного чтения,
    3. Для количественного чтения.
  5. Степени детализации:
    1. Интегральные,
    2. Детальные.

Характеристики зрительного анализатора (учитываются при констурировании СОИ):

  1. Энергетические (диапазон воспринимаемых яркостей, контрастность, относительная видимость).
  2. Информационные (пропускная способность).
  3. Пространственные (острота зрения, объем восприятия, поле зрения).

    4. Поле зрения – пространство зрительного восприятия при неподвижной голове и фиксированном взгляде.

    Зоны поля зрения:

    • Центральное зрение (до 4°),
    • Зона ясного видения (30° - 35°),
    • Зон периферического зрения (75° - 90°).
  4. Временные:
    1. Латентный период реакции,
    2. Длительность инерции ощущения,
    3. Время адаптации,
    4. Длительность информационного поиска.

Использование акустического анализатора целесообразно:

  • Если информация простая, краткая, требует медленной реакции,
  • При перегрузке визуального анализатора,
  • Если необходима голосовая связь.
